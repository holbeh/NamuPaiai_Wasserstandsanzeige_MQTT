21 // A servo is activated by creating an instance of the Servo class passing
22 // the desired pin to the attach() method.
23 // The servos are pulsed in the background using the value most recently
24 // written using the write() method.
25 //
26 // The methods are:
27 //
28 // Servo - Class for manipulating servo motors connected to Arduino pins.
29 //
30 // attach(pin ) - Attaches a servo motor to an i/o pin.
31 // attach(pin, min, max ) - Attaches to a pin setting min and max values in microseconds
32 // default min is 544, max is 2400
33 //
34 // write() - Sets the servo angle in degrees. (invalid angle that is valid as pulse in microseconds is treated as microseconds)
35 // writeMicroseconds() - Sets the servo pulse width in microseconds
36 // read() - Gets the last written servo pulse width as an angle between 0 and 180.
37 // readMicroseconds() - Gets the last written servo pulse width in microseconds. (was read_us() in first release)
38 // attached() - Returns true if there is a servo attached.
39 // detach() - Stops an attached servos from pulsing its i/o pin.
