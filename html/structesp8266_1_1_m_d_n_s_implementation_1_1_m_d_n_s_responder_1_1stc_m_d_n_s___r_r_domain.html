MDNSResponder::stcMDNS_RRDomain

A MDNS domain object. The labels of the domain are stored (DNS-like encoded) in 'm_acName': [length byte]varlength label[length byte]varlength label[0] 'm_u16NameLength' stores the used length of 'm_acName'. Dynamic label addition is supported. Comparison is supported. Export as byte array 'esp8266.local' is supported.
